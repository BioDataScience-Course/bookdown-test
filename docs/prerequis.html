2.1 Prérequis

Si ce n’est déjà fait (matériel du module 1), vous devez installer et vous familiariser avec la ‘SciViews Box’, RStudio et Markdown. Vous devez aussi maitriser les bases de git et de Github (avoir un compte Github, savoir clôner un dépôt localement, travailler avec Github Desktop pour faire ses commits, push et pull).
